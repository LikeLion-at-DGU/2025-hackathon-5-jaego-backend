## ­ЪќЦ ьћёвАюВаЮьіИ ВєїЖ░ю

в│И Вёюв╣ёВіцвіћ ВХЕвг┤вАю ВДђВЌГВЮў ВєїВЃЂЖ│хВЮИЖ│╝ Вєїв╣ёВъљвЦ╝ ВЌ░Ж▓░ьЋўвіћ **вАюВ╗г ВъгЖ│а вДѕВ╝Њ ьћївъФьЈ╝**ВъЁвІѕвІц.

ВаљВБ╝Ж░ђ вѓеВЮђ ВъгЖ│авЦ╝ ВЏљЖ░ђв│┤вІц ьЋаВЮИвљю Ж░ђЖ▓ЕВЌљ вЊ▒вАЮьЋўвЕ┤, Вєїв╣ёВъљЖ░ђ ВъгЖ│а ВДђвЈё, ьі╣Ж░ђ ВЃЂьњѕ, ai вДъВХц ВХћВ▓ю, Ж▓ђВЃЅ ЖИ░віЦ вЊ▒ВЮё ьєхьЋ┤ ьјИвдгьЋўЖ▓ї ВЃЂьњѕВЮё В░ЙЖ│а ВўѕВЋйьЋа Вѕў ВъѕвЈёвАЮ вЈЋвіћ Вёюв╣ёВіцВъЁвІѕвІц. 

ВЮ┤вЦ╝ ьєхьЋ┤ ВєїВЃЂЖ│хВЮИВЮђ ьЈљЖИ░ в╣ёВџЕВЮё ВаѕЖ░љ + ВХћЖ░ђ ВѕўВЮх В░йВХю ьџеЖ│╝ВЌљ вЇћвХѕВќ┤ вДцВъЦ в░ЕвгИВю╝вАю ВєїВєїьЋю ьЎЇв│┤ ьџеЖ│╝вЦ╝ Вќ╗Ж│а, Вєїв╣ёВъљвіћ ьЋЕвдгВаЂ Вєїв╣ё вІѕВдѕвЦ╝ ВХЕВА▒ьЋўвЕ░ ВДђВЌГ Вѓгьџї вѓ┤ ВЌ░Ж▓░ВЮё Ж▓гЖ│аьъѕ ьЋўвіћ ьџеЖ│╝вЦ╝ ЖИ░вїђьЋа Вѕў ВъѕВіхвІѕвІц.

## ­Ъџђ ВБ╝Вџћ ЖИ░віЦ

- **ьџїВЏљЖ░ђВъЁ/вАюЖиИВЮИ**: Вєїв╣ёВъљ / ьїљвДцВъљ ЖхгвХё
- **вДцВъЦ вЊ▒вАЮ**: Step1(ВБ╝Вєї/ВаЋв│┤) Рєњ Step2(ВёювЦў ВЌЁвАювЊю)
- **ВЃЂьњѕ Ж┤ђвдг**: вЊ▒вАЮ / ВѓГВаю, ьЋаВЮИВюе, ВюаьєхЖИ░ьЋю
- **Вєїв╣ёВъљ ЖИ░віЦ**:
    - Жи╝В▓ў Ж░ђЖ▓ї ВЃЂьњѕ в░Ј Ж░ђЖ▓ї Ж▓ђВЃЅ (ВюёвЈё/Ж▓йвЈё ЖИ░в░ў, В╣┤ьЁїЖ│авдг ЖИ░віЦ ьЈгьЋе)
    - ВъгЖ│а ВДђвЈёвАю ьїљвДц ВцЉВЮИ Жи╝В▓ў ВЃЂьњѕ ьЃљВЃЅ (ЖИ░в│И 5kmВЮ┤вѓ┤ ВЃЂьњѕВЮђ вЈёьєавдгвАю ьЉюВІю, ВЮ┤ВЎИ вѓўвГЄВъј)
    - ВўѕВЋй в░Ј ВиеВєї (ВъгЖ│а ВъљвЈЎ в░ўВўЂ)
    - В░ю, ьі╣Ж░ђ ВЃЂьњѕ(30%), ВХћВ▓ю ВЃЂьњѕ ьЎЋВЮИ
    - ВБ╝вгИ вѓ┤ВЌГ ьЎЋВЮИ(ВБ╝вгИВЃЂьЃю, ВБ╝вгИ ВЃЂВёИ, вДцВъЦВюёВ╣ў ВДђвЈёВЌљВёю ьЎЋВЮИ Ж░ђвіЦ)
- **ВўѕВЋй Ж┤ђвдг**:
    - ВЃЂьЃю ВаёьЎў: `pending Рєњ confirm Рєњ ready Рєњ pickup / cancel`
    - ВЋївд╝(Notification) ВъљвЈЎ ВЃЮВё▒
- **ВъљвЈЎьЎћ ЖИ░віЦ**:
    - ВўѕВЋй 10вХё В┤ѕЖ│╝ ВІю ВъљвЈЎ ВиеВєї (Celery + Redis)
    - ВюаьєхЖИ░ьЋю вДївБї ВІю ВЃЂьњѕ ВъљвЈЎ в╣ёьЎюВё▒ьЎћ
    - ВХћВ▓ю Въёв▓авћЕ в▓Аьё░ ВъљвЈЎ Ж░▒ВІа
      
Рађ

## ­ЪЏа в░░ьЈг ьЎўЖ▓й / ВѓгВџЕ ЖИ░Вѕа

- **Вёюв▓ё Вџ┤ВўЂВ▓┤Ваю**: Ubuntu 22.04 LTS
- **в░░ьЈг ВБ╝Вєї**: `yeonhee.shop`
- **ВЏ╣ Вёюв▓ё**: Nginx + Gunicorn
- **в░▒ВЌћвЊю**: Django REST Framework, SimpleJWT
- **вЇ░ВЮ┤ьё░в▓аВЮ┤Віц**: SQLite
- **в╣ёвЈЎЖИ░ ВъЉВЌЁ / Task Queue**: Celery + Redis
    - Redisвіћ AWS ElasticacheВЌљВёю ьўИВіцьїЁ
- **AI ВХћВ▓ю вАюВДЂ**: OpenAI Embedding API (`text-embedding-3-small`) ВѓгВџЕ
- **ВБ╝Вџћ в░░ьЈг/ьўИВіцьїЁ**: AWS EC2
- **ВаЋЖИ░ ВъЉВЌЁ**: cron / Celery BeatвЦ╝ ВЮ┤ВџЕьЋ┤ ВЃЂьњѕ Въёв▓авћЕ в░Ј ВўѕВЋй/ВюаьєхЖИ░ьЋю В▓ўвдг
  
Рађ

## ­ЪЊА API ВЌћвЊюьЈгВЮИьіИ

- `/accounts/` : ьџїВЏљЖ░ђВъЁ, вАюЖиИВЮИ, вАюЖиИВЋёВЏЃ, вѓ┤ ВаЋв│┤
- `/stores/` : вДцВъЦ вЊ▒вАЮ(step1/step2), ВА░ьџї, Вўцьћѕ/вДѕЖ░љ
- `/products/` : ВЃЂьњѕ вЊ▒вАЮ, ВѓГВаю, ВА░ьџї, В░ю, ьі╣Ж░ђ, Жи╝В▓ў ВЃЂьњѕ ВА░ьџї
- `/reservations/` : ВўѕВЋй ВЃЮВё▒, ВЃЂьЃю в│ђЖ▓й(confirm/cancel/ready/pickup)
- `/notifications/` : ВЋївд╝ ВА░ьџї в░Ј ВЮйВЮї В▓ўвдг

## ­Ъћё ВўѕВЋй ьћёвАюВёИВіц

1. Вєїв╣ёВъљЖ░ђ ВЃЂьњѕ ВўѕВЋй Рєњ ВЃЂьЃю `pending`
2. ьїљвДцВъљЖ░ђ ВўѕВЋй ьЎЋВЮИ Рєњ ВЃЂьЃю `confirm`, ВўѕВЋй ВІюЖ░ё ЖИ░вАЮ
3. ьїљвДцВъљЖ░ђ ВЃЂьњѕ Вцђв╣ё ВЎёвБї Рєњ ВЃЂьЃю `ready`
4. Вєїв╣ёВъљЖ░ђ ьћйВЌЁ ВЎёвБї Рєњ ВЃЂьЃю `pickup`
5. ВиеВєї ВІю: ВѓгВюа ЖИ░вАЮ + ВъгЖ│а в│хЖхг + ВЋївд╝ ВЃЮВё▒
- ВўѕВЋй ВаёВєА 10вХё ВЮ┤вѓ┤ в»ИВЮЉвІх ВІю ВъљвЈЎ ВиеВєї
   
Рађ

### ­ЪДа AI ВХћВ▓ю ВІюВіцьЁю Ж░юВџћ

в│И ьћёвАюВаЮьіИвіћ OpenAI Embedding API(`text-embedding-3-small`)вЦ╝ ьЎюВџЕьЋўВЌг **ВЃЂьњѕ ВХћВ▓ю ВІюВіцьЁю**ВЮё ЖхгьўёьќѕВіхвІѕвІц.

### 1. Въёв▓авћЕ ВЃЮВё▒

- ВЃЂьњѕвфЁ, ВЃЂВаљвфЁ, В╣┤ьЁїЖ│авдгвфЁВЮё ьЋўвѓўВЮў вгИВъљВЌ┤вАю вгХВќ┤ OpenAI Embedding APIВЌљ ВъЁваЦ
- Ж▓░Ж│╝вАю вѓўВўе **1536В░еВЏљ в▓Аьё░**вЦ╝ `NumPy` в░░ВЌ┤вАю ВађВъЦ
    - `product_ids.npy` : ВЃЂьњѕ ID в░░ВЌ┤
    - `product_vectors.npy` : ВЃЂьњѕ в▓Аьё░ в░░ВЌ┤

### 2. ВѓгВџЕВъљ в▓Аьё░ ВЃЮВё▒

- Вєїв╣ёВъљЖ░ђ В░юьЋю ВЃЂьњѕвЊцВЮў в▓Аьё░вЦ╝ вХѕвЪгВў┤
- ВхюЖи╝ nЖ░ю(`max_recent`) ВЃЂьњѕ в▓Аьё░вЦ╝ Ж░ђВцЉ ьЈЅЖиаьЋўВЌг **ВѓгВџЕВъљ в▓Аьё░** ВЃЮВё▒

### 3. ВХћВ▓ю ВЋїЖ│авдгВдў

- ВйћВѓгВЮИ ВюаВѓгвЈё ЖИ░в░ўВю╝вАю ВѓгВџЕВъљ в▓Аьё░ВЎђ ВЃЂьњѕ в▓Аьё░ в╣ёЖхљ
- вЈЎВЮ╝ В╣┤ьЁїЖ│авдг / вЈЎВЮ╝ ВЃЂВаљ / Ж░ђЖ╣їВџ┤ Ж▒░вдг(ьЋўв▓ёВѓгВЮИ Ж▒░вдг Ж│ёВѓ░)ВЌљ вћ░вЮ╝ в│┤вёѕВіц ВаљВѕў вХђВЌг
- ВхюВбЁ ВаљВѕўЖ░ђ ВЮ╝ВаЋ threshold ВЮ┤ВЃЂВЮИ ВЃЂьњѕвДї ВХћВ▓ю
